# Changelog

本文档记录 GreenPrj 各版本的变更。版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

---

## [v0.1.0-web-stable]（未发布）

**标签**：`v0.1.0-web-stable`  
**对应**：Web 端首个稳定基线版本。

### 已实现的主要功能

- **认证与安全**
  - 本地密码设置、登录、修改
  - 数据加密（AES-256-GCM）
  - 会话管理（默认 30 分钟无操作自动退出）

- **核心记账**
  - 手动录入账目（金额、日期、类别、备注）
  - 账目列表查看、编辑、删除
  - 按时间区间、类别、关键词的过滤与搜索

- **统计与报表**
  - 按时间（天/周/月/年）与类别的消费统计
  - 趋势图、类别分布图

- **预算管理**
  - 月度/年度预算设置
  - 预算执行情况与超支提醒

- **数据管理**
  - JSON / CSV 导出
  - 备份导入、冲突检查与合并
  - 安全删除与数据清除

- **账本合并**
  - 导出/导入账本
  - 去重与冲突解决

- **操作日志**
  - 关键操作记录、查看、筛选与导出

- **离线支持**
  - 纯前端本地存储（IndexedDB），可完全离线使用

### 已知限制

- Web 版本为**本地单机版**，数据不跨设备同步；需通过导出/导入手动迁移。
- 不支持密码找回；忘记密码只能清除本地数据后重新开始（请提前导出备份）。
- 当前无云端后端；未来若接入云服务会在文档与 UI 中明确说明。

### 后续计划

- Android 端认证与记账功能对齐、跨端账本导出/导入闭环验证
- Web 生产部署流水线、性能与前端监控
- 跨端术语与体验一致性优化

# Product Brief: GreenPrj

## Executive Summary

GreenPrj 是一款专注于隐私安全和个人数据主权的跨平台个人记账应用（Android/iOS/Web）。与市面上依赖云端同步的记账APP不同，GreenPrj采用本地优先的数据存储策略，确保用户的财务数据完全掌控在自己手中。通过智能识别技术（发票扫描、短信解析、语音输入）和多端账本合并功能，帮助用户轻松实现"随时记录、随时掌握"的记账体验，同时为未来可能的云端服务演进预留架构空间。

## Core Vision

### Problem Statement

个人财务管理中，用户面临的核心痛点是：

1. **记账及时性问题**：消费后容易遗忘或遗漏，导致账目不完整，无法准确掌握资金流向
2. **数据隐私担忧**：现有记账APP大多依赖云端同步，用户担心财务数据泄露或被第三方滥用
3. **多端数据割裂**：用户在手机和PC端分别记账，数据分散，难以统一管理和统计
4. **记账操作繁琐**：手动输入效率低，缺乏智能识别和自动化能力
5. **统计维度单一**：现有APP在时间维度（周/月/年）和类别维度（衣食住行等）的统计不够灵活和直观

### Problem Impact

- **资金管理失控**：不知道钱花在哪里，无法合理规划消费额度
- **隐私风险**：财务数据上传云端，存在数据泄露、被分析、被出售的风险
- **使用体验差**：记账成为负担而非工具，用户难以坚持长期使用
- **决策支持不足**：缺乏多维度统计分析，无法为消费决策提供有效参考

### Why Existing Solutions Fall Short

**主流记账APP的局限性：**

1. **随手记**：
   - 优点：功能专业完善，多平台同步，用户基数大
   - 缺点：界面陈旧，有广告骚扰，数据存储在云端，隐私风险高

2. **薄荷记账**：
   - 优点：功能性与简洁性兼顾，无广告
   - 缺点：UI美观度不足，操作较繁琐，同样依赖云端同步

3. **MoneyWiz**：
   - 优点：国际化专业记账，功能大而全
   - 缺点：价格较高，主要面向海外用户

4. **本地记账APP（如卡皮）**：
   - 优点：隐私保护强，数据完全本地存储
   - 缺点：无法多端同步，设备丢失风险高，功能相对简单

**核心差距：**
- 缺乏"隐私安全 + 多端支持 + 智能识别 + 账本合并"的综合解决方案
- 现有APP要么牺牲隐私换取便利，要么牺牲便利换取隐私
- 没有充分考虑用户在不同设备上的记账场景和账本合并需求

### Proposed Solution

GreenPrj 提供一套**本地优先、隐私至上、多端协同**的个人记账解决方案：

**第一阶段（本地记账工具）：**

1. **多平台支持**：
   - Android/iOS移动端：支持快速记账、语音输入、发票扫描、短信/微信回执识别
   - Web端：支持PC端记账和详细数据管理

2. **智能记账**：
   - 发票/账单扫描识别消费信息
   - 手机消费短信自动解析
   - 微信消费回执识别
   - 语音输入记账
   - 用户确认后自动入账

3. **灵活统计**：
   - 时间维度：按周/月/年统计消费趋势
   - 类别维度：衣食住行等分类占比分析
   - 预算对比：实际消费与预算的对比分析

4. **账本合并**：
   - 支持手机端账本导入PC端，或PC端账本导入手机端
   - 智能去重：自动识别并合并重复账目（日期、项目、金额一致）
   - 统一汇总统计管理

5. **数据安全**：
   - 所有数据本地存储，不上传云端
   - 支持数据导入导出（便于备份和迁移）
   - 用户完全掌控自己的财务数据

**第二阶段（架构预留）：**
- 架构设计考虑未来可能的云端同步需求
- 支持个人多端账本统一管理
- 为后续功能演进（如数据分析、AI建议等）预留扩展空间

### Key Differentiators

1. **隐私安全优先**：
   - 本地存储策略，数据不上传云端，避免隐私泄露风险
   - 符合《个人信息保护法》《数据安全法》要求
   - 用户完全掌控数据，可随时导出和删除

2. **智能识别能力**：
   - 发票/账单扫描识别
   - 短信/微信回执自动解析
   - 语音输入支持
   - 降低记账操作成本，提高及时性

3. **多端账本合并**：
   - 独特的账本合并功能，解决多端数据割裂问题
   - 智能去重算法，避免重复统计
   - 统一汇总管理，提供完整财务视图

4. **架构前瞻性**：
   - 第一阶段本地优先，满足隐私需求
   - 架构设计预留云端演进空间
   - 支持未来功能扩展（多端同步、数据分析等）

5. **跨平台一致性**：
   - Android/iOS/Web三端统一体验
   - 适配不同使用场景（移动端快速记账、PC端详细管理）
   - 数据格式统一，便于导入导出和合并

### User Journey

#### 发现阶段
- **渠道**：朋友推荐、朋友圈看到、家人介绍
- **触发点**：看到朋友分享的消费统计图表，或听到朋友推荐"这个记账APP很安全，数据都在本地"
- **第一印象**：隐私安全、本地存储、多端支持

#### 首次使用（Onboarding）
1. **下载安装**：从应用商店下载Android/iOS版本，或访问Web端
2. **密码设置**：首次使用设置登录密码，保护数据安全
3. **快速上手**：
   - 尝试快速记账功能
   - 体验语音输入或扫描发票
   - 查看统计报表示例
4. **价值感知**：看到清晰的消费分类和时间统计，感受到产品的实用性

#### 日常使用（Core Usage）
- **消费后记录**：
  - 购物后立即用手机快速记账
  - 使用智能识别（发票扫描、短信解析）减少手动输入
  - 语音输入提高记账效率
- **定期查看**：
  - 每天查看当日消费
  - 每周回顾周度统计
  - 每月分析月度消费趋势和类别占比
- **预算管理**：
  - 设置月度预算
  - 查看预算执行情况
  - 根据统计调整消费计划

#### 价值感知时刻（Aha Moment）
- **第一次发现消费规律**：通过统计报表发现"原来我在XX类别上花了这么多钱"
- **成功控制超支**：通过预算对比功能，成功将月度消费控制在预算内
- **账本合并成功**：将手机和PC端的账本合并，看到完整的财务视图
- **隐私安全感**：意识到数据完全在本地，密码保护，不用担心隐私泄露

#### 长期使用（Habit Formation）
- **养成记账习惯**：每天消费后习惯性打开APP记录
- **定期分析**：每周/每月定期查看统计，分析消费趋势
- **持续优化**：根据统计数据不断优化消费习惯和预算规划
- **推荐他人**：感受到产品价值后，主动推荐给有记账需求的朋友和家人

#### 多端协同使用
- **移动端**：主要用于快速记账和随时查看
- **Web端**：用于详细数据管理、完整报表分析和账本合并
- **数据同步**：通过导入导出功能，在手机和PC端之间同步账本数据

## MVP Scope

### Core Features

**MVP核心功能清单：**

#### 1. 用户认证与数据安全
- **密码登录**：用户设置密码保护账本数据
- **数据加密**：本地数据加密存储
- **数据安全**：确保端侧财务数据安全

#### 2. 记账功能
- **手动记账**：用户手动输入消费信息（金额、日期、类别、备注等）
- **发票扫描**：扫描发票/账单，自动识别消费信息，用户确认后入账
- **短信解析**：读取手机消费短信，自动解析消费信息，用户确认后入账
- **语音输入**：语音输入记账，自动识别并转换为文字，用户确认后入账
- **消费分类**：支持衣食住行等消费类别管理

#### 3. 统计功能
- **时间维度统计**：按周/月/年统计消费趋势
- **类别维度统计**：按类别（衣食住行等）统计消费占比
- **基本统计报表**：消费趋势图表、类别占比图表等

#### 4. 预算管理
- **预算设置**：用户可以设置月度/年度预算
- **预算对比**：实际消费与预算的对比分析
- **预算提醒**：预算超支提醒功能

#### 5. 账本管理
- **账本合并**：支持Android端账本导入Web端，或Web端账本导入Android端
- **智能去重**：自动识别并合并重复账目（日期、项目、金额一致）
- **统一统计**：合并后的账本统一汇总统计管理

#### 6. 数据管理
- **数据导入**：支持导入账本数据
- **数据导出**：支持导出账本数据（便于备份和迁移）
- **本地存储**：所有数据本地存储，不上传云端

#### 7. 平台支持
- **Android平台**：完整的Android移动端应用
- **Web平台（Windows）**：完整的Web端应用，支持Windows系统

### Out of Scope for MVP

**明确不在MVP范围内的功能：**

1. **云端同步**：
   - 多端实时同步功能
   - 云端数据备份
   - 多设备统一管理
   - **计划**：第二阶段实现

2. **高级数据分析**：
   - 深度财务分析
   - 消费预测
   - 财务健康评分
   - **计划**：后续版本实现

3. **AI建议**：
   - 智能消费建议
   - 个性化理财建议
   - 消费习惯分析
   - **计划**：后续版本实现

4. **iOS平台**：
   - iOS移动端应用
   - **计划**：MVP验证成功后实现

5. **其他高级功能**：
   - 多账户管理
   - 投资理财功能
   - 借贷管理
   - 家庭共享账本
   - **计划**：根据用户反馈和需求逐步实现

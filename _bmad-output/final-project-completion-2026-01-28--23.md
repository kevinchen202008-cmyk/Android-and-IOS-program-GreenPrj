## 技术实现总结

### Web平台技术栈

- **UI框架**: Material UI (MUI)
- **状态管理**: Zustand
- **数据库**: IndexedDB (idb 3.0+)
- **加密**: AES-256-GCM
- **密码哈希**: bcryptjs
- **OCR**: Tesseract.js
- **语音识别**: Web Speech API
- **测试**: Playwright (E2E), Vitest (Unit)

- **语言**: Kotlin
- **架构**: Clean Architecture + MVVM
- **依赖注入**: Hilt (Dagger)
- **数据库**: Room 2.8.4
- **UI框架**: Material Design Components
- **异步**: Kotlin Coroutines + Flow
- **构建工具**: Gradle
